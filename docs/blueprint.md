# **App Name**: Solvia

## Core Features:

- Transparent Navbar: A transparent navbar that becomes slightly solid on scroll, with smooth background fade and subtle link hover animations.
- Hero Section: A hero section with a product name (SOLVIA), headline, supporting text, and primary/secondary CTAs. Includes a large editorial illustration with subtle animations.
- Video Section: A centered video container with a clean frame and subtle hover/animation effects. Includes a play button with subtle pulsing animation.
- About Section: An 'About' section with a short intro paragraph and bullet points, featuring subtle reveal and slide-in animations on scroll.
- Features Section: A 'Features' section with feature cards in a clean grid, including subtle hover lift, icon animations, and upward card animation on scroll entry.
- Pricing Section: A transparent pricing section with two pricing cards, a monthly/yearly toggle, and subtle animations for price changes and CTA buttons.
- Contact Form: A clean form for lead capture, minimal background contrast, and input focus highlights.

## Style Guidelines:

- Primary color: Soft Blue (#A8D0E6) to reflect the subtle blue accent and create a calm and trustworthy feel.
- Background color: Very light blue (#F0F8FF), almost white, to create a clean and spacious layout.
- Accent color: Light grayish-blue (#BBD0D7), used sparingly for micro-interactions and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral feel.
- Minimal editorial style icons: Bold outlines with flat fills, simple geometry, anthropomorphic icons
- Single-page scrolling layout with clear section separation and smooth scroll navigation.
